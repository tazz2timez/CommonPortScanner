# 🔎 Python Port Scanner

A fast and simple multithreaded port scanner built with Python. This tool scans a target IP or domain to detect open ports, identify services, and perform basic banner grabbing.

---

## 🚀 Features

- 🔍 Scans common ports or custom port ranges
- ⚡ Multithreaded for faster performance
- 🌐 Supports both IP addresses and domain names
- 🧠 Service detection (HTTP, SSH, etc.)
- 🏷️ Basic banner grabbing
- ⏱️ Timeout handling for reliability
- 💾 Save results to a file
- 🖥️ Interactive mode + CLI support

---

## 🛠️ Technologies Used

- Python 3
- `socket` (network connections)
- `argparse` (CLI arguments)
- `concurrent.futures` (multithreading)
